# Mini Slot (PixiJS + TypeScript)

## Requirements

- Node.js 20+
- npm

## Installation

```bash
git clone https://github.com/USERNAME/mini-slot.git
```

```bash
cd mini-slot
```

```bash
npm install
```

## Run

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

## Features

- 5x3 slot grid
- Spin button
- Balance
- Current bet
- Win display
- Winning symbols highlight
- Scatter bonus
- JSON-driven spins
